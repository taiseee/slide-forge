# DESIGN — デザイン基盤

slide-forge のデザイントークンと書式ルール。実装の正は `theme/core.css` と各スキンにあり、
本書はエージェント・人間が参照するための要約。**ここにない装飾は使わない。**

## スライドサイズ・セーフエリア

- サイズ: **16:9(1280×720px)のみ**。4:3 / A4 は未対応(必要になったら ROADMAP で判断)
- 本文セーフエリア: 上60px / 左右84px / 下66px(`section` の padding)
- ヘッダー: 右上(top 26px)、フッター: 左下(bottom 28px)、ページ番号: 右下
- グリッドは持たない。列構成はレイアウトクラス(`columns` `comparison` 等)が決める

## タイポグラフィ

- 見出し・本文: Hiragino Kaku Gothic ProN → Helvetica Neue → Noto Sans JP
- 大きな数字: Avenir Next(`--sf-numeral-font`、weight 200–300 で軽く)
- 基準サイズ: 25px。階層は em で統一

| 用途 | サイズ | ウェイト・字間 |
|---|---|---|
| スライドタイトル(h1) | 1.3em | 600 / 0.09em |
| 小見出し(h2) | 1em | 600 / 0.08em |
| 本文 | 1em | 400 / 0.03em、行間 1.7 |
| 注釈・出典 | 0.8em 前後 | muted 色 / 字間広め |
| ヘッダー・フッター | 13px | 500 / 0.16em |
| 大きな数字(stat/kpi) | 2.9–4.4em | 200–300 |

- 箇条書きの階層: 1段目 `–`、2段目 `·`(0.9em)。**ネストは1段まで、1スライド5項目まで**
- 強調は `**strong**`(accent-strong 色・600)のみ。下線・イタリック・色文字での強調はしない
- 引用・出典は `>` blockquote(左罫線+薄パネル)。リンクは accent 色+細下線
- 数式は KaTeX(`math: katex`)、コードはモノスペース+薄背景

## カラートークン

構造(core.css)は色を持たず、以下の変数をスキンが上書きする。

| 変数 | 役割 | research | business | lecture | soft |
|---|---|---|---|---|---|
| `--sf-bg` | 背景 | #ffffff | #fbfaf7 | #fbfcfa | #fdfdfc |
| `--sf-text` | 本文 | #2f3237 | #38332c | #2f342f | #3a3735 |
| `--sf-muted` | 補足・番号 | #878d96 | #98907f | #8a9088 | #9b948c |
| `--sf-line` | ヘアライン | #e3e5e9 | #e6e0d4 | #e1e5df | #eae6e1 |
| `--sf-panel` | 薄い帯 | #eef0f3 | #f1ece2 | #edf1ea | #f3f0ec |
| `--sf-accent` | アクセント | #566173 | #8a7a63 | #5c7360 | #c4705c |
| `--sf-accent-strong` | 強アクセント | #343c48 | #55483a | #3b4c40 | #8e4a3e |
| `--sf-ink` | 濃色面 | #343a44 | #3d372e | #37423a | #514741 |
| `--sf-ink-text` | 濃色面上の文字 | #f5f6f8 | #f7f5f0 | #f4f6f2 | #faf6f3 |

soft スキン(グレージュ×コーラル)のみ角丸トークンを持つ: `--sf-radius: 14px`(パネル・箱・画像)/
`--sf-radius-s: 8px`(コードブロック)/ `--sf-radius-pill: 999px`(チップ・中心ノード)。
また soft は blockquote の左罫線を持たない(角丸パネルのみで引用・注記を表現する)。

(実値は `theme/research.css` / `theme/business.css` / `theme/lecture.css` が正。ずれていたら CSS に合わせる)

## グラフ用カラーパレット(作図時に使う)

グラフ・図は CSS では描かず画像を貼る(SKILL.md 参照)。作図する際は以下の色に限定する。

- **research**: 主系列 `#566173` / 強調 `#343c48` / 補助 `#9aa3af` `#c7ccd4` / 対比 `#a8927a`
- **business**: 主系列 `#8a7a63` / 強調 `#55483a` / 補助 `#b5a88f` `#d9d2c3` / 対比 `#6e7b8a`
- **lecture**: 主系列 `#5c7360` / 強調 `#3b4c40` / 補助 `#9aab9c` `#ccd6cc` / 対比 `#7a6e8a`
- **soft**: 主系列 `#c4705c` / 強調 `#8e4a3e` / 補助 `#d8ab9e` `#e8ddd6` / 対比 `#64837b`
- 状態を示す記号色(表の ○×△、注意書き等に最小限): 成功 `#5e7a5e` / 警告 `#a8863c` / エラー `#9c5450` / 情報 `#566173`
- 背景はスキンの `--sf-bg` に合わせ、彩度の高い色・虹色の多色使いはしない

## 装飾ルール(禁止事項)

エディトリアル・ミニマルを守るため、以下は**使わない**:

- グラデーション、ドロップシャドウ、強い角丸(角丸はコード片の3pxのみ。
  例外: `soft` スキンは角丸トークンで統一的に丸める設計のため、この項目の対象外)
- カードUI(枠+影の箱)。区切りはヘアライン(1px の `--sf-line`)で行う
- 太いバー・帯・塗りつぶしの多用。濃色面は takeaway / lead / 各レイアウトの強調1箇所まで
- 絵文字・装飾アイコンの多用(checklist の ✓ 等、レイアウトが定義するものだけ)

## 画像・写真のトーン

- 写真: 彩度低め・モノトーン寄りが馴染む。`logos` はグレースケール表示される
- 図・ダイアグラム: 背景は白か `--sf-panel`、線色・塗りは上のパレットから選ぶ
- 背景画像: 写真は `![bg right:45%](...)`(cover)、図は `![bg right:45% fit](...)`

## ヘッダー・フッター・ページ番号

- ページ番号: `paginate: true`(表紙・終了は `<!-- _paginate: false -->`)
- 機密区分・Draft: `header: CONFIDENTIAL`(全体)または `<!-- _header: DRAFT -->`(1枚)→ 右上
- 所属・コピーライト: `footer: © 2026 Example Inc.` → 左下
- 表紙・区切りで消したい場合は `<!-- _header: "" -->` / `<!-- _footer: "" -->`
